# QuickMeet-App
It is a Zoom like video call app made using Node.JS, Express.JS, Socket.IO, WebRTC, React.JS

In this app a user can call another user by a unique ID and start video calling.

### To start this app on your machile, Do the following steps.

1. open your browser
2. copy paste this url: https://quickmeet-souvik-dey.netlify.app/
3. enjoy video calling...
